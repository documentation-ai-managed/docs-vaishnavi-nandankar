## Migration to v1.2.0

Follow these steps to upgrade safely.

<Steps>
  <Step title="Backup" icon="database">
    Export your documentation site:
    
````bash
npx docs-export --site ./docs --output backup.zip
````
    
  </Step>
  <Step title="Update Dependencies" icon="package">
    <CodeGroup tabs="npm,yarn">
````bash
npm install docs-platform@1.2.0
````
````bash
yarn add docs-platform@1.2.0
````
    </CodeGroup>
  </Step>
  <Step title="Run Migration" icon="settings">
    
````bash
npx docs-migrate v1.2.0
````
    
    Review changes in `.changeset/`.
  </Step>
</Steps>
